## Особенности

    Визуальный выбор атрибутов: Цветные кружки для выбора цвета и кнопки для размеров

    Динамическое обновление: Мгновенное изменение цены, остатков и изображения

    Валидация комбинаций: Блокировка недоступных вариантов

    Кэширование: Оптимизация производительности через WordPress Transients

    AJAX взаимодействие: Быстрое добавление в корзину без перезагрузки

    Адаптивный дизайн: Корректное отображение на всех устройствах

    Без плагинов: Чистая кастомная разработка

## Безопасность

    Nonce проверки для всех AJAX запросов

    Санитизация входных данных

    Валидация комбинаций перед добавлением в корзину

    Экранирование выводимых данных

## Требования

    WordPress 5.0+

    WooCommerce 4.0+

    PHP 7.4+

    jQuery

## Установка

    склонируйте репозиторий в папку wc-variations дочерней темы WordPress
    git clone https://github.com/shamalli/wp-wc-variations.git

    добавьте строчку в functions.php дочерней темы
    `include_once "wc-variations/functions.php";`

    копировать api-data.json в корень WordPress сайта

    редактировать api-data.json, в соответствии с ID продуктов WooCommerce

## Поиск неисправностей

### Селектор не отображается

    Проверьте наличие данных в JSON файле

    Убедитесь, что product_id совпадает с ID товара

    Проверьте ошибки в консоли браузера

### Данные не обновляются

    Проверьте AJAX запросы в Network вкладке

    Убедитесь в правильности nonce

    Проверьте структуру JSON данных

### Не добавляется в корзину

    Проверьте валидность комбинации

    Убедитесь, что товар в наличии

    Проверьте WooCommerce настройки корзины
